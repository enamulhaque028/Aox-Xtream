# aos_xtream

Aos Xtream

#### Installation

In the command terminal, run the following commands:

    $ git clone https://github.com/enamulhaque028/Aox-Xtream.git
    $ cd Aox-Xtream/
    $ flutter run

To run on Web(chrome)
    $ flutter run -d chrome --release
